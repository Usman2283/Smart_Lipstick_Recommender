# Smart_Lipstick_Recommender
I built a Smart Lipstick Recommender, a full-stack web application that analyzes a user's skin tone from an uploaded photo and suggests personalized lipstick shades. The project leverages Flask as the backend web framework to handle API requests and serve the frontend interface. For image processing and skin detection, I used OpenCV (cv2) along with NumPy to perform operations like reading images, converting color spaces (BGR to RGB), extracting the center region for face detection, and calculating average RGB values. The skin tone classification is done using a brightness-based rule engine: if the average brightness exceeds 150, the skin tone is classified as Fair; between 100 and 150 as Medium; and below 100 as Dark. Based on the detected tone, the system returns four curated lipstick shades with their hex color codes and descriptions. Although the project originally included a TensorFlow deep learning model trained using scikit-learn for train-test splitting and h5py for model serialization, I ultimately switched to a more reliable rule-based approach due to the small synthetic dataset. On the frontend, I used HTML5, CSS3, and vanilla JavaScript to create a responsive drag-and-drop upload interface with real-time image preview and asynchronous server communication via the Fetch API. Additional libraries include Werkzeug for secure filename handling, SciPy as a dependency, and Python's venv for environment management. The result is a fully functional beauty tech application that demonstrates the integration of computer vision, web development, and machine learning concepts into a practical real-world solution.
